# Проєкт: Форма реєстрації користувача

## 1. Функціональні та нефункціональні вимоги

### Функціональні вимоги (Що робить сторінка):
* Користувач може ввести своє ім'я, email, пароль та підтвердження пароля.
* Користувач може вибрати свою роль (Розробник, Дизайнер, Менеджер) через випадаючий список.
* При натисканні на кнопку "Зареєструватися" відбувається перевірка (валідація) полів через JavaScript.
* Якщо паролі не збігаються або email некоректний — форма не відправляється, а під полями виводяться червоні помилки.
* При успішній валідації на екрані з'являється зелене повідомлення про успіх.

### Нефункціональні вимоги (Технічні властивості):
* Сторінка має чисту семантичну HTML5 розмітку.
* Усі стилі винесені в окремий зовнішній файл `css/style.css` і прив'язані виключно через класи.
* Інтерактивність реалізована через події JavaScript у файлі `js/main.js`.
* Сторінка адаптивна — коректно відображається як на моніторах, так і на мобільних телефонах.

## 2. Схема дизайну сторінки (Page design schema)
Інтерфейс сторінки має вертикальну блочну структуру центровану по центру екрану:
```
+-------------------------------------------------------+
|                       HEADER                          |
|                "Створення аккаунту"                   |
+-------------------------------------------------------+
|                        MAIN                           |
|  +-------------------------------------------------+  |
|  |                   SECTION                       |  |
|  |  [ Поле: Ім'я ]                                 |  |
|  |  [ Поле: Email ]                                |  |
|  |  [ Поле: Пароль ]                               |  |
|  |  [ Поле: Підтвердження пароля ]                 |  |
|  |  [ Випадаючий список: Роль ]                    |  |
|  |                                                 |  |
|  |  [ Кнопка: Зареєструватися ]                    |  |
|  +-------------------------------------------------+  |
+-------------------------------------------------------+
|                       FOOTER                          |
|             "© 2026 Усі права захищено"               |
+-------------------------------------------------------+
```
